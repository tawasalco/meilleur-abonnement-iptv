- `pricing/` - Pricing plans

## Deployment

### Cloudflare Pages (Recommended)

1. Push to GitHub
2. Connect repository to Cloudflare Pages
3. Set build command: `npm run build`
4. Set output directory: `dist`
5. Deploy

Or via CLI:

```bash
npm run deploy
```
